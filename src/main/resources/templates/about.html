About CRISPRts

Application

This tool allows easy design of sgRNA oligos, which contains a 5' T7 promoter and a complementary sequence for PCR amplification with a scaffold oligo. This PCR product is then used as template for in vitro transcription. This tool can be also used to download DNA sequence files from the Ensembl Genome Browser, with annotated exons.
